# HR Analytics Capstone Project – IOTBTECH Data Analytics Program
This repository contains my **HR Analytics Capstone Project**, completed as part of the **IOTBTECH Data Analytics Program**. The project demonstrates an end-to-end data analytics workflow in **Microsoft Excel**, beginning with a raw HR dataset exported from a legacy Human Resource Management System (HRMS) and transforming it into a clean, structured, and analysis-ready dataset.

The project involved comprehensive **data profiling**, **data cleaning**, and **data transformation**, including duplicate removal, handling missing values, standardizing data formats, validating records using lookup tables, and enriching the dataset with calculated fields such as **Full Name**, **Department Name**, **Years of Service**, **Performance Band**, and **Eligible Bonus Amount**. Advanced Excel functions including **XLOOKUP**, **DATEDIF**, **IF**, **COUNTIF**, **COUNTIFS**, **AVERAGEIF**, and **SUMIF** were used to automate calculations and improve data quality.

Following the data preparation stage, I performed **Exploratory Data Analysis (EDA)** using Pivot Tables to answer key HR business questions, including employee headcount by department, average salary analysis, departmental attrition rates, employee performance comparisons, and projected bonus payouts. The findings were then presented through an interactive **HR Dashboard** featuring KPI cards, Pivot Charts, and Slicers, enabling users to explore workforce metrics dynamically.

This project highlights practical skills in **Data Cleaning**, **Data Transformation**, **Data Validation**, **HR Analytics**, **Exploratory Data Analysis (EDA)**, **Dashboard Development**, **Business Intelligence**, **Data Visualization**, and **Microsoft Excel**. It also demonstrates the importance of transforming raw operational data into meaningful insights that support strategic workforce planning and data-driven decision-making.
